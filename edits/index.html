DDR X2 Edit Creator

Click to see instructions
Instructions:
  1. Create edits for charts you want to play in Stepmania, using files from Zenius-i-Vanisher as a base. (IMPORTANT: Note the restrictions listed below)
  2. Click the "Upload .SM/.SSC Files" button to upload up to 30 edits (drag and drop is also supported)
  3. For each chart:
    1. Verify that the correct song is selected from the drop down. If the song you want is not in the list, it is not playable on DDR X2 US arcade cabinets
    2. Select the correct chart within your simfile that you want to play on the machine
    3. Select the difficulty (e.g. Expert, Challenge, etc) and meter (the numerical rating from 0-20) that you want for your edit
    4. Type an optional name for your edit. Names can contain up to eight characters, and can only contain capital letters, numbers, spaces, and the following special characters: .!@#$%&-_=+\()
    5. Optionally, select up to two phrases to appear below your chart in game.
  4. At the bottom, select the region of the machine you intend to play your edits on. (This must match or the edits will not load)
  5. When you are satisfied with your charts, press "Create Edit."
  6. If you have any errors, they will be displayed below. If the problem is with the chart, you can remove/reupload it to fix the problem. (Changing the file locally will not automatically reupload the changes to the website)
  7. If there are no errors, you will get a download for DDR_EDIT_X.DAT, where X corresponds to the region you selected. Save this file in a folder called "DDR_EDIT" on a flash drive. For example, if the flash drive is mounted at Z:, the correct path for this file would be Z:\DDR_EDIT\DDR_EDIT_X.DAT. Do not change the filename.
Restrictions:
  1. Only 12th notes and 16th notes are allowed (this includes 4ths and 8ths, but does not include 24ths, 32nds, etc)
  2. Shock arrows are not allowed
  3. You cannot have 12ths and 8ths/16ths in the same beat
  4. You cannot have have three or more notes start at the same time
  5. A freeze cannot end exactly one 12th or 16th note before another begins, on the same arrow
  6. On double, you cannot have stretch jumps. This includes all of the following:
    1. L---L---
    2. L----D--
    3. L-----U-
    4. L------R
    5. -D---D--
    6. -D----U-
    7. -D-----R
    8. --U--D--
    9. --U---U-
    10. --U----R
    11. ---R---R
  7. Warning: Edits to ΔMax will crash US (and possibly European) DDR X2 machines.
